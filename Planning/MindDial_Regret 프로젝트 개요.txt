MindDial_Regret 프로젝트 개요

앱 이름 : 마인드 다이얼_후회
기획 배경 : 후회라는 감정에 매몰되어 고통받는 사람들에게 도움을 주고싶어서 개발을 계획하게 되었다.
기획 목적 : 현실에 치이는 현대인들을 위해 후회가 주는 부정적인 감정은 줄이고 교훈과 긍정적인 영향을 극대화 시키는 감정조절 메뉴얼을 제공해 준다.
타겟 사용자 : 10살 이상 감정의 압도되는 예민한 감수성을 가진 사용자
주요 기능 : 후회란 무엇인지 감정을 분석하고 CBT기법을 이용한 감정 조절과 매일마다 실천할법한 일일 미션을 제공해준다.
개발 플렛폼 : Android 모바일
개발 도구 : React Expo, FireBase 기반 서버리스 개발
출시 예정일 : 2026년 4월 22일
